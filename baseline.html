MULTIMODAL FEATURES ENCODING

Following MISA, one-dimensional semantic features are previously extracted from each utterance or audio/video clip, then projected into a modality-specific space utilizing feature projection model Ep. Ep is pretrained on a multimodal emotion recognition task for better understanding in multimodal correlation. The textual/acoustic/visual features are encoded by Ep to form multimodal conversation context input ut,a,v.
